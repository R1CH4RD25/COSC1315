Learning Objectives

By the end of this lesson, students will be able to:

Create and use variables in Python

  • Define variables to store values using the assignment operator (=).

  • Understand that variables are like labeled containers that hold data.

  • Assign different types of values to variables (strings, integers, floats, booleans).

  • Use descriptive variable names that clearly indicate the data they store.

Follow Python naming conventions

  • Use snake_case for variable names (lowercase with underscores: first_name).

  • Start variable names with a letter or underscore (never with a number).

  • Avoid Python reserved keywords (like print, if, for) as variable names.

  • Choose meaningful names (price is better than p or x).

Understand variable assignment and reassignment

  • Recognize that = means "assign" not "equals" (it stores a value in a variable).

  • Update variable values by assigning new values (count = 5 then count = 10).

  • Understand that the previous value is replaced when reassigning.

Work with different data types

  • Store integers (whole numbers) in variables: age = 16

  • Store floats (decimal numbers) in variables: price = 19.95

  • Store strings (text) in variables: name = "John"

  • Store booleans (True/False) in variables: is_active = True

Display and use variable contents

  • Print variable values using print(variable_name) without quotes.

  • Combine variables with text in output statements.

  • Understand that printing the variable name (no quotes) displays its value.

  • Recognize the difference between print("x") (prints "x") and print(x) (prints value of x).

Debug variable-related errors

  • Fix NameError when a variable is used before being defined.

  • Correct misspelled variable names.

  • Ensure variables are defined in the correct order before use.
