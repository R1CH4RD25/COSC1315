üí° Tips for Success

  • Remember the colon! Every if, elif, and else statement needs a colon (:) at the end

  • Check your indentation: Code inside if statements must be indented (4 spaces)

  • Use == for comparison: Remember that = assigns values, while == compares values

  • Test your code: Run each cell after writing code to see if it works

  • Read error messages: Python tells you exactly what's wrong - read the error carefully!
