üìö Key Terms

Arithmetic Operator
A symbol that performs a mathematical operation on numbers (e.g., +, -, *, /).
Operator Precedence
The rules determining the order in which operators are evaluated in an expression. Also known as the order of operations.
Floor Division (//)
Division that rounds down to the nearest integer, discarding any decimal portion (e.g., 10 // 3 equals 3).
Modulus (%)
An operator that returns the remainder after division (e.g., 10 % 3 equals 1).
Exponentiation (**)
An operator that raises a number to a power (e.g., 2 ** 3 equals 8).
Augmented Assignment
Operators like += and -= that combine an arithmetic operation with variable assignment (e.g., x += 5 is equivalent to x = x + 5).
Parentheses
Symbols () used to group expressions and control the order of evaluation, overriding default precedence rules.
Math Module
A built-in Python library that provides advanced mathematical functions like ceil(), floor(), sqrt(), and trigonometric functions.
Binary Operator
An operator that works with two operands (e.g., 5 + 3, where + is the binary operator).
PEMDAS
A mnemonic for remembering order of operations: Parentheses, Exponents, Multiplication/Division, Addition/Subtraction. In Python, exponentiation (**) is the exponent operation.
