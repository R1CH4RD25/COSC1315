Learning Objectives

By the end of this lesson, students will be able to:

Write and execute basic Python programs

  • Use the print() function to display text and numbers on the screen.

  • Create and run Python code in Google Colab by typing code in cells and pressing Shift+Enter.

  • Understand that Python programs execute line-by-line from top to bottom.

Work with strings and quotes

  • Define strings using single quotes ('text') or double quotes ("text").

  • Recognize that Python treats single and double quotes identically for simple strings.

  • Handle strings containing apostrophes or quotes by using alternate quote types or escape sequences.

Understand Python syntax basics

  • Explain that Python is case-sensitive (print works, Print does not).

  • Ensure parentheses are balanced and quotes come in matching pairs.

  • Use comments (#) to document code and explain what each section does.

Display different types of data

  • Print text (strings) by enclosing content in quotes: print("Hello")

  • Print numbers (integers) directly without quotes: print(42)

  • Create multiple lines of output using multiple print() statements.

Debug common beginner errors

  • Fix NameError by correcting function name capitalization.

  • Fix SyntaxError by ensuring quotes and parentheses are properly matched.

  • Read error messages to identify and resolve simple coding mistakes.
