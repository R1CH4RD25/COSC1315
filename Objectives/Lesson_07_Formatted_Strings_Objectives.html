  • Locate substrings using .find() method

  • Replace text using .replace() method

  • Check if text exists with the in operator

  • Recognize that string methods return new strings

  • Understand that original strings are never modified

  • Assign method results to variables when needed

Recognize common pitfalls

  • Remember to prefix with f before the opening quote

  • Use curly braces correctly: {variable} not $variable

  • Avoid forgetting parentheses on methods: .upper() not .upper

  • Understand that methods don’t change the original string

üîë Key Terms

  • Formatted String (f-string) — A string prefixed with f that allows embedding expressions inside curly braces

  • String Concatenation — Joining strings together using the + operator

  • Placeholder — A position in a formatted string marked by {} that gets filled with a value

  • String Method — A function that belongs to string objects and performs operations on them

  • .upper() — Method that returns a new string with all letters in uppercase

  • .lower() — Method that returns a new string with all letters in lowercase

  • .title() — Method that capitalizes the first letter of each word

  • .strip() — Method that removes leading and trailing whitespace from a string

  • .find() — Method that returns the index of the first occurrence of a substring (-1 if not found)

  • .replace() — Method that returns a new string with specified text replaced
