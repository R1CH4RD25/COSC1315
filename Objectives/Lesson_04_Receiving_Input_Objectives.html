Learning Objectives

By the end of this lesson, students will be able to:

Get user input in Python programs

  • Use the input() function to prompt users for information.

  • Display a custom prompt message to guide user input: input("What is your name? ")

  • Store user responses in variables for later use in the program.

  • Create interactive programs that respond to user input.

Understand how input() works

  • Recognize that input() always returns data as a string (text).

  • Understand that the program pauses and waits for the user to type and press Enter.

  • Explain that whatever the user types becomes the return value of input().

Create interactive programs

  • Ask the user for their name and display a personalized greeting.

  • Prompt for multiple pieces of information (name, age, favorite color, etc.).

  • Build programs that collect data and display customized responses.

  • Combine input() with print() to create conversational programs.

Work with user-provided data

  • Store input in variables: name = input("Your name: ")

  • Use input values in subsequent calculations or output.

  • Recognize that numeric input needs type conversion for math operations (covered in Lesson 05).

Format output with user input

  • Display user input as part of a larger message or sentence.

  • Create multi-line programs that prompt, store, and display information.

  • Use descriptive prompts that clearly tell users what to enter.

Debug input-related issues

  • Fix errors caused by missing parentheses or quotes in input() calls.

  • Ensure input is stored in a variable before trying to use it.

  • Understand the difference between input("message") and print("message").
